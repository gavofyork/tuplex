# Motivation

For tuple's type conversions.

# Usage

Add this crate to Cargo.toml

`Cargo.toml`:

```toml
tuplex = "0.1"
```

# Features

1. adding/removing element at front/back

2. converting heterogeneous tuples to homogeneous ones

# Document

See the [doc](https://docs.rs/tuplex) for more examples.
